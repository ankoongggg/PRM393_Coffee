# PRM393 Coffee — Flutter scaffold

This repository now contains a minimal Flutter project scaffold.

Quick start (Windows / PowerShell):

1. Install Flutter SDK: https://flutter.dev/docs/get-started/install
2. From this repository root, run:

```powershell
flutter pub get
flutter run
```

Notes:

- Use `flutter devices` to check connected devices or emulators.
- If you open the project in Android Studio / VS Code, the IDE will prompt to install required plugins.

Files added:

- `pubspec.yaml` — dependencies and metadata
- `lib/main.dart` — minimal Flutter app (Hello, Flutter!)
- `.gitignore` — common Flutter ignores

Next suggestions:
